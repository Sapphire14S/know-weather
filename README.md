# Know Weather

Know Weather is a React-based weather application that provides real-time weather information through a clean and responsive interface.

## Live Demo

https://sapphire14s.github.io/know-weather/

## Features

* Search weather by city name
* Real-time weather data
* Temperature, humidity, and wind information
* Loading and error handling states
* Responsive user interface
* Fast and intuitive experience

## Tech Stack

* React
* Vite
* JavaScript
* CSS
* Weather API

## Installation

```bash
git clone https://github.com/Sapphire14S/know-weather.git
cd know-weather
npm install
npm run dev
```
